EXR

 

Purpose

 

To create a .EXR file or an .EXR enumerated sequence of files.

In the particular case that the source files were also .exr,  then all the source layers can be preserved (by using the All Layers mode).

Inputs

 

AV In: The image track or tracks to be encoded.

 

Ext. Audio (Optional): The audio tracks to be encoded. If audio codec is activated, these tracks will provide the audio content for transcoding.

 

Outputs

 

AV Out: A .EXR file or or enumerated sequence of files and an audio file if the audio codec property is activated.

Node specific properties

 

Transcoding->Codec: The EXR format provides many possible variants:

 

Choose RGBA instead of RGB if your exr files have to include alpha channel

 

Choose Zip or DWA depending on the compression type that you want. DWA applies a JPEG-like lossy compression (with much better quality than JPEG) while Zip does a lossless compression.  However, EXR DWA can also produce lossless files depending on the situation, producing smaller files that are faster to playback.

 

The EXR  format does not have  any resolution limits (apart from hardware capabilities).

 

When transcoding from other formats only the main RGB/A channels are rendered (multilayer is not supported for other formats). But if you choose any of the options labeled as All Layers and the original source clips loaded upstream are also EXR files then the new exr files will also preserve all the layers found in the source files.

 

 

You can watch a video tutorial using this node here

 

Note: You can extract layers from exr multilayer files trough the EXR input node, and then pass it to this node (or to any other transcoder node).

Note: You can preserve all the layers from source exr files by selecting one of the All layers modes in the codec settings. But what is not currently possible is to insert new layers to exr output files.

Note: The encoding speed of this task with is very CPU intensive (except the "RAW" variants, which are uncompressed and only affected by the storage speed).
