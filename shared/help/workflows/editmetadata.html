Edit Metadata

 

Purpose

 

Permits adding or changing user metadata associated with the input media, including technical data about camera recording settings, camera & lenses description, production metadata, and slate metadata.  You can also create custom metadata to add keywords or user comments.  

This node also permits to create sidecar metadata files (in the same folder and with same base name as the original media, but different extension). Mistika supports two formats for sidecar metadata files: .XMP and .CSV, which are widely supported by other post production applications such as Resolve, Pomfort Silverstack or the Adobe suite. You can select the preferred format on the Export Metadata property, or leave it as InFile for using embedded metadata only.  Those metadata files can also be created with transcoder nodes (so you only need this node if you want to change user metadata content).

Some alternatives to this node are the Set UP Metadata (which will overwrite the metadata of one file with the metadata from another one) and the NC to metadata node, which can extract metadata values directly from the file names of the input files.

This video provides a general introduction about how to manage metadata in Mistika Workflows

 

Note that this node can only change "user metadata" fields, other metadata fields need to maintain coherence with media files and can only be manipulated with specialized nodes (codec metadata, color metadata, etc).

Node specific properties

 

Export Metadata(InFile, .CSV, XMP):  Select file format for sidecar metadata files, to be created when executing this node (no extra nodes are needed at the output if you just want to create the metadata files ). The InFile option will just pass the metadata to posterior nodes, for example to be used in the transcoder nodes or as naming tokens (As seen in the tutorial video mentioned above).

 

USER METADATA PROPERTIES:

 

Settings Information: Recording settings including Shutter Speed, Focal Length, White Balance, Tint, Fstops, ISO.

 

Camera, Lens: Camera and Lens descriptions, including manufacturer, Model, Serial Nr, Device Id, Firmware Version.

Production: Director, Production, Producer, Cinematographer, Camera Assistant, Camera Assistant 2, DIT, Data Manager, Script Supervisor, Sound Mixer, Location, Copyright.

Slate Information: Season, Episode Nr, Episode Name, Scene, Shot, Take, Slate Camera, Slate Description, Int/Ext, Day/Night. Shooting Date

Custom: to add extra keywords and comments to the metadata information.

NOTE: When creating metadata files, in addition to user metadata managed by this node it will also include other media metadata found / added by previous nodes, even if not having  dedicated properties in this node.
