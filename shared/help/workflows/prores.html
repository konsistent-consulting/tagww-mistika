ProRes

 

Purpose

 

Create .mov media files in Apple ProRes format

Inputs

 

AV In: The image track to be encoded.

 

Ext. Audio (Optional): The audio tracks to be encoded. Alternatively, you can use embedded audio from movie files connected to the AV In connector, which is decided with the Use AV audio property) .

 

Outputs

 

AV Out: A video file with .mov extension, containing all the video and audio tracks.

Node specific properties

 

Transcoding->Codec: You can choose one of the following, ProRes Proxy, ProRes LT, ProRes, ProRes HQ, ProRes 4444, Prores4444 + Alpha, ProRes XQ, ProRes XQ + Alpha, according to your needs.

 

ProRes Proxy is an even more highly compressed codec than Apple ProRes LT, intended for use in offline workflows.

 

ProRes LT is a more highly compressed codec than Apple ProRes 422, with roughly 70 percent of the data rate and 30 percent smaller file sizes.

 

ProRes is a high-quality compressed codec offering nearly all the benefits of Apple ProRes 422 HQ, but at 66 percent of the data rate for even better multistream, real-time editing performance.

 

ProRes HQ is a higher-data-rate version of Apple ProRes 422 that preserves visual quality at the same high level as Apple ProRes 4444 but for 4:2:2 image sources.

 

ProRes 4444 features full-resolution, mastering-quality 4:4:4:4 RGB color and visual fidelity that is perceptually indistinguishable from the original material.

 

Prores4444 + Alpha is the same as ProRes 4444 but including alpha channel.

 

ProRes XQ is the highest-quality version of Apple ProRes for 4:4:4:4 image sources.  This format has a very high data rate to preserve the detail in HDR.

 

ProRes XQ + Alpha is the same as ProresXQ but including alpha channel.
