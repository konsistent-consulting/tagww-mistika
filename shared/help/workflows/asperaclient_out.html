Aspera Client

 

Purpose

 

File transfer through the Aspera platform. This node allows you to upload or download files (depending if you use the Input or Output node versions).

 

Outputs (input nodes only)

 

To: Any type of downloaded content from the Aspera Server will be connected through the "To" output.

 

Inputs (output nodes only)

 

File: The file to be transferred.

 

 

Standard properties

Object Name

Color

Bypass

Lock

 

Node specific properties

 

User: Type here the username of your account.

 

Certificate: You can login by using a ssl cartificate credentials. Locate the certificate by clicking on the "..." button.

 

Password: Type the password to access to your account.

 

Local file path (Input nodes only): Local file path is a property for the input Aspera node where you have to assign the folder where the downloaded files will be placed.

 

Remote file path: It is the server destination address and it must contain the complete path, including folders and subfolders where the content to download is located (Input node version), or the exact location where you want to upload the files (Output node version).

 

TCP port: TCP port for the connection.

 

UDP port: UDP port for the connection.

 

Target Rate (Mbps): Tarjet transmission speed. Reduce it to avoid using too much bandwidth of your Internet connection.

 

Retries: If the connection is lost, how many times will be retried without exiting the workflow processing with an error.

 

Rename Files List: To apply renaming rules for special characters while uploading files, in the form of comma separated pairs like Character1SubstitutionString1, Character2SubstitutionString2, and so on

 

For example, let's suppose your client does not accept uploads using the regional characters "ñ" and "ç" in their names, so you decide to substitute them by "NN" and "CC" instead. Then you will put this rule:  ÑNN,ÇCC

 

Or if you just want to remove those characters do not put any substitution string.  In the above example, you would put this instead: Ñ,Ç
