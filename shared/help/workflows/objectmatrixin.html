Object Matrix In

 

Purpose

 

These nodes allow to upload or download media files to Object Matrix cloud services.

 

Inputs (output nodes only)

 

in: This property selects the content to be sent via FTP. This content can be any structure of files organized in folders and subfolders.

 

 

Outputs (input nodes only)

 

out: Downloaded content from the Object Matrix server is stored in Local Path property location and then passed to the out connector.

 

Standard properties

Object Name

Color

Bypass

Lock

Name Convention

 

Python standard properties

code

This node is written in Python, and the source code is provided for additional flexibility.

 

 

Node specific properties

 

Bucket: The Object Matrix bucket to download files from.

 

Subfolders are supported. Just add /folderName at the end of the bucket name. For example:

 

Bucket:  MyBucket1/MyFolder3

 

Note that adding a "/" at the end of the bucket path means to download its content, not the folder itself. For example, if your local folder is D:/MyDownloads:

 

Bucket: MyBucket1/MyFolder3  

It will create a local folder D:/MyDownloads/MyBucket1/MyFolder3 (if not existing yet), and then it will download the contents to it

 

Bucket: MyBucket1/MyFolder3/  

It will download all the contents from MyBucket1/MyFolder3 directly  into the local folder D:/MyDownloads/ , without creating any folder on it.

 

 

endPointURL: End point URL provided by Object Matrix.

 

Local Path (input node only): Where to store downloaded files.  

 

New Files Only: Only  download new files not existing in the destination folder.

 

Delete after download: Delete the files from the bucket after downloading them.

 

Note: In addition to the media files, these nodes can be customized to support automatic pass trough for their associated Object Matrix own metadata, as long as Object Matrix In and Object Matrix Out nodes are connected to a same workflow. Examples are provided in the Python code property. Alternatively please contact support if you need custom modifications.
