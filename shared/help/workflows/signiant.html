Signiant

 

Purpose

 

File transfer through the Signiant platform.

 

Inputs

 

File: The file to be transferred.

 

 

Standard properties

Object Name

Color

Lock

 

Node specific properties

 

User: Type here the username of your account.

 

Password: Type the password to access to your account.

 

Source Agent: Write the name of the agent from where the transfer is made.

 

Target Agent: Enter the name of the agent who will receive the transfer.

 

Target Directory:  Signiant folder to upload files to.

 

URL: URL needed to make the transfer.

 

Job Template Library:  job template to use, (Media_Mover_Workflows by default)

 

Job Group: Identify the Group of members.

 

Auto Schedule: Submit the job to Signiant auto scheduler
