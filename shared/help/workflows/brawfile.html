BRAWFile

 

Purpose

 

This node will load BRAW files and set their properties before passing them to other nodes.

 

Outputs

 

AV Out: The BRAW files specified in the File property with the metadata settings for the image extraction defined in this node.

 

 

Inputs

 

Raw: Input to connect BRAW files.

Node specific properties

 

The next properties will only appear when Debayer is set to custom mode:

 

Gamma, Gamut: Drop-down menu to select BRAW Gamma and Gamut. By default they will pass trough the original settings ( --- From File --- value), so they will remain as "unknown" until processing the incoming files. Alternatively, you can force a particular image extraction from the raw camera files with other values.

These particular properties are BRAW specific, but depending on the selection they will also set the gamma and gamut properties of the node and they will be displayed below.

 

Change ISO, Change Kelvin, Change Tint,  Change Exposure, HighLight Recovery,  3D LUT, IPP2 Saturation, Contrast, Shadows, MidPoints, HighLight, Black Level, White Level, Video Black: These properties will use the metadata settings from the BRAW file by default, but they can be forced to other values here.

ReduceFactor: Default value is 1. You can set a division factor to make an image extraction at reduced resolution, which is much faster than extracting the images at full resolution and then reducing the resolution in a posterior transcoder node.
