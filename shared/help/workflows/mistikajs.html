Mistika JS

 

 

Purpose

 

To create Mistika .js image files using Mistika own format. The Mistika .js is an image format optimized for full quality (uncompressed) and high performance in Mistika applications (real-time performance up to 8K 60p uncompressed ).

This format does not have  any resolution limits (apart from your hardware capabilities).

 

Inputs

 

AV In: The image track to be encoded.

 

Ext. Audio (Optional): The audio tracks to be encoded.

 

 

Outputs

 

AV Out: An image file with .js extension containing the input video track.  Optionally, it can also create a companion audio file in the selected audio format ( .js files do not support embedded audio).

YUV444 10b

YUV 16b

 

 

Troubleshooting

 

If you miss some metadata in the .js media file (ie timecodes etc) don't look in there, as it will be in a different place. This is because the Mistika .js node normally requires it to be used in combination with the Mistika Mlnk task node, which creates the project metadata file describing it, and which is also needed to import this type of media in Mistika applications.

 

Note: Being an uncompressed format, the encoding speed of this task node mainly depends on disk speed, so it is highly recommended to use it with fast storage units (NVMe or disk arrays).
