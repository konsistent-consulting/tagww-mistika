Crop

 

Purpose

 

To crop the input media while creating black area (or solid color) around the cropped area. Permits cropping all channels  (RGBA) or selectively (RGB / A). It also provides softness control, and an optional color extrusion mode to fill the cropped area.

Note that this node will not render a new version by itself, instead it will define the cropping operation and it will tell posterior nodes with render capability to apply it when rendering new files.

Note that what you control with this node is what happens with the cropped areas, but the output image will have the same resolution as the original. This node can not define a new output resolution by itself.  If what you want is to obtain a lower resolution image with only the center-crop area then you can achieve that with any transcoder task node directly. Transcoder nodes will center-crop to a definable output resolution, you just need to disable their fit property to discard the cropped parts.

 

Inputs

 

Input_1: Files to be cropped.

 

Outputs

 

Output: Input files, with the new crop instructions applied.

 

 

Standard properties

Color

Node specific properties

 

Mode: To select which channel or channels will be affected by the crop: RGB, RGBA or Alpha. The optional Extrude mode crops the image using a extrusion to the borders.

 

Crop: The percentage of the image to be extruded from Left, Right, Up or Down.

 

Soft: The percentage of the image to be softened from Left, Right, Up or Down once cropped.

 

Color: Percentage of color to be applied to the cropped parts.
